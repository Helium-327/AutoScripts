# VSCode创建工作区的脚本

## 功能：

- [ ] 用户自定义保存路径
  - [ ] 添加路径
    - [ ] 工作区文件夹路径
    - [ ] 工作区配置文件路径
  - [ ] 修改路径
    - [ ] 工作区文件夹路径
    - [ ] 工作区配置文件路径
- [X] 创建工作区
- [X] 打开已有工作区
- [X] 删除工作区

## 实现方案

- 整体功能可以使用 `python`、`batch`、`lua`等脚本语言实现
- 路径设置可以使用 `config`或者 `env`来实现，判断是否存在文件，存在读取，不存在则创建
- 每个一级功能均由函数实现
